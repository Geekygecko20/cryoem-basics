# Cryo-EM Basics

Exploring fundamental math and physics concepts behind Cryo-EM image formation.

### Topics
- Fourier transforms
- Contrast transfer function (CTF)
- Noise & filtering
- 2D image alignment

### Example Notebooks
| Notebook | Description |
|-----------|--------------|
| `fourier_transform.ipynb` | Visualizing 2D Fourier transforms |
| `ctf_simulation.ipynb` | Simulating CTF under different defocus values |

### How to Run
```bash
git clone https://github.com/Geekygecko20/cryoem-basics.git
cd cryoem-basics
jupyter notebook notebooks/fourier_transform.ipynb
